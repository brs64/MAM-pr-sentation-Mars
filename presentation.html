Architecture & Déploiement

Services Docker Consolidés (7 conteneurs)

  • btsplay-laravel - Application PHP-FPM
  • btsplay-queue - Worker de transcodage asynchrone
  • btsplay-nginx - Reverse proxy (port 8080)
  • btsplay-mysql - Base de données MySQL 5.7
  • btsplay-phpmyadmin - Administration DB (port 8001)
  • btsplay-ftp-pad - FTP pour MXF (port 21001)
  • btsplay-ftp-arch - FTP pour MP4 (port 21002)

Volumes persistants : dbdata, ftp-pad-data, ftp-arch-data, /mnt/archivage, /mnt/miniatures

Démonstration Live

Application en action sur localhost:8080

💡 Pour lancer : docker compose up

Sécurité & Conformité - Parcours D

Sécurité Application

  • Isolation des services Docker
  • Hachage des mots de passe (bcrypt)
  • Eloquent ORM anti-injection SQL
  • CSRF protection Laravel
  • Spatie Permissions (rôles granulaires)

Sécurité Infrastructure

  • VPN pour accès distant sécurisé
  • Politiques de sécurité Windows Server
  • Pare-feu et ports restreints
  • FTP authentifié (Pure-FTPd)
  • Nginx reverse proxy

Conformité RGPD : Gestion des données personnelles, consentements, droit à l'oubli

Sobriété Numérique & Éco-conception

Génération Asynchrone

Jobs en arrière-plan - Miniatures générées à la demande uniquement

Lazy Loading

Chargement progressif - Ressources chargées uniquement si visibles

Compression Images

Miniatures JPEG optimisées - 10x moins lourdes que les PNG

Streaming Adaptatif
